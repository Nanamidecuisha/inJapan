1. 名1 は　名2 より 一类形/二类型 です。
	比较名词1 和 名词2 ，名词1 更具有句中形容词所表示的性质时使用
	A は　B より　xxです。（A 比 B 更xx）

	李さん　は　森さん　より　若いです。（小李比 森先生更年轻）
	中国　は　日本より　広いです。（中国比日本更宽阔）
	上海 は　北京　より　にぎやかですか。（上海比北京 更繁荣）
	程度的差异比较大的时候 用副词　ずっと强调。
	中国　は　日本　より　ずっと　広いです。（中国比日本辽阔得多）

2. 名1 より　名2　の　ほうが 一类形/二类型　です。
	比较名词1 和 名词2 ，名词2 比名词1 更具有句中形容词所表示性质时使用这个句型。
	A は　B の　ほうが　xxです。（与A相比，B更 xx）

	日本　より　中国の　ほうが　広いです。（与日本相比，中国更辽阔）
	北京　より　東京の　ほうが　暖かいです。（与北京相比，东京更暖和）
	大阪　より　東京の　ほうが　賑やかですか。（与大阪相比，东京更热闹吗？）

3. 名1 は　名2　ほど 一类形　くないです/ 二类型　では　ありません。
	A は　B ほど　xx (形容词的否定形式) 意思是：A 比 B更不 xx

	東京の冬　は　北京の冬　ほど　寒いくないです。（东京的冬天没有北京的冬天那么冷）
	神戸は　大阪　ほど　賑やかでは　ありません。（神户没有大阪那么热闹）
	私の部屋　は　王さんの部屋　ほど　綺麗では　ありません。（我的房间没有小王的房间那么漂亮）

4. 名1 の　中で　名2　が　一番 一类形/二类型　です。
	在某种范围中，名词2 更具有某种性质。

	スポーツの　中で　サッカー　が　一番　面白いです。（在各种体育活动中，足球最有意思）
	日本料理の　中で　寿司が　一番　美味しいです。（日式饭菜中，寿司最好吃）

	但是 名1 表示 场所，时间 则后面不能加 の中　而使用
	场所/时间 で　一番　一类形/二类型 名2 は 名3　です。

	日本で　一番　高い　山は　富士山です。（在日本最高的山是富士山。）
	一年で　一番　寒い　季節は　二月です。（一年当中最冷的季节是二月份。）

5. 名1 と　名2 と　どちらが　一类形/二类型 ですか。
	询问两种事物当中 哪一种更具有形容词的性质时，使用这个句型。どちら　是两者选一的疑问句，回答时用　〜の　ほうが　〜　如果两者程度相当，则使用　どちらも　〜です，どちら有时在口语中有时会说成　どっち。

	日本語と　英語と　どちらが　難しいですか。（ 日语和英语 哪个更难）
	日本語　の　ほうが　難しいです。（日语更难）

	コーヒー　と　紅茶　と　どっちが　好きですか。（咖啡喝红茶，更喜欢哪个）
	紅茶　の　ほうが　好きです。（更喜欢红茶）
	どちらも　好きです。（都喜欢）

	询问两种事物用这个语法。

6. どの　名词/　いつ　/　どれ　/ 誰　/　何 が一番　一类形/二类型 ですか。
	询问三种以上事物哪个更有某种性质时，用这个语法。
	询问人 ： 誰が
	询问物 ： どの /　どれ /　何　+　が
	询问时间 ： いつ　が

	この　中で　どの　料理が　一番　美味しいですか。（这些菜里面哪个菜最好吃）
	この　料理が　一番　美味しいです。（这个菜最好吃）

	季節　の　中で　いつが　一番　好きですか。（四个季节当中，你最喜欢哪个季节）
	私　は　春が　一番　好きです。（我最喜欢春天）

	被选择的事物已见于上下文或存在于眼前，使用　どの　〜　が或 どれ　が，反之 用 何が

	リンゴと　ナシと　バナナ　の中　で　どれが　一番　好きですか。
	果物の　中で　何が　好きですか。

7. 最近， 表示从较近的过去到现在为止的一段时间，与中文不同，表示过去时。

8. xx が　いいです
	いい　中文表示好

	箱根　の　お土産　は　何が　いいですか。（箱根的礼品什么好）
	寄木細工が　いいです。（木片拼花工艺品比较好）
	私は　肉より　魚の　ほうが　いいです。（比起肉来我更喜欢吃鱼）

9.やっぱり
	やっぱり 是 やはり 比较随便的说法，用于正常谈话，表示某信息或事态的发展与自己预测的一致，意思是 正如自己所预料的一样。

	やっぱり　雨が　降りましたね。　（下雨前猜测或已经知道要下雨，果然下雨了呢）
