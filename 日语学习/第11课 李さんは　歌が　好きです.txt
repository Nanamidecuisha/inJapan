1. 名1 は 名2　が　一类形/二类形容词 です
	好きです　嫌いです　怖いです　等表示情感的形容词都可以。
	名1 表示情感主体，名2 表示情感对象

	小野さん　は　歌が　好きです。（小野先生喜欢唱歌。）
	李さん　は　肉が　嫌いです。（李先生讨厌肉。）
	林さん　は　犬が　怖いです。（林先生害怕狗。）

2. 名1 は 名2が　分かります/できます
	分かります　できます　上手です　下手です　苦手です 等有关能力的词，也可以 
	名1 は　名2　が 动词的　ます　形 /一类形容词 或 二类形容词　です

	スミスさん　は　韓国語が　分かります。（史密斯先生能明白韩语。）
	私は　コンピユータ　が　できません。（我不会电脑）
	小野さん　は　料理が　上手です。（小野先生擅长做饭。）

3. 名　や 名
	助词 や　和 と　一样，用于连接名词，两者的区别在于　と 用于列举所有的项目，而 や　用于只列举许多项目中的两项。并且　や 常常和　なと 呼应使用。相当于中文中的 ~什么和~什么 等等...

	吉田さん　は　時々　中国や　韓国へ　行きます。（吉田先生 有时候 会去中国 韩国）
	森さん　は　サッカー　や　野球が　好きです。（森先生 喜欢 足球和棒球）
	私は　お店で　野菜や　果物なとを　買いました。（我在店里买了蔬菜 水果等等。）

4. から 和　だから（理由原因）
	说明理由或原因用助词　から。放在原因的后面
	一类形容词 + 动词 ~から　　　暑いから　　食べますから

	森さん　は　お酒が　好きです　から、毎日　飲みます。（因为森先生喜欢喝酒，所以每天都喝）
	寒いですから、窓　を　閉めます。（因为很冷，关上了窗）
	窓を　閉めます。寒いですから。（关了窗，因为很冷）

	だから 表示原因理由的连词，意思相当于汉语中的所以。比较礼貌的用法 是ですから 。だから〜 放在后果的前面，句首。因为什么原因，所以什么（后果）
	二类形容词 + 名词 〜だから/是ですから 	好きですから

	キムさん　は　野菜が　嫌いです。だから　あまり　野菜を　食べません。（金先生讨厌蔬菜，所以他不吃）
	明日は休みです。ですから、子供と　動物園へ　行きます。（明天休息，所以和孩子一起去动物园）

5. 表示频率的副词
	いつも　よく　時々　たまに　あまり　全然　等是表示频率的副词，あまり　全然 后面要用否定形式。

	私　は　たまに　コンサートへ　行きます。（我 偶尔会去看音乐会）
	李さん　は　時々　映画を　見ます。（李先生 有时会去看电影）
	吉田さん　は　あまり　横浜へ　行きません。（吉田先生 几乎没有去过横滨）

6. どしてですか
	询问事由时使用 どしてですか，相当于汉语中的为什么。回答时再谓语后面加 から。

	僕は　来月　また　ここへ　来ますよ。
	どしてですか。
	ここで　友達　の　結婚式が　ありますから。
7. ホテル　で　写真展が　あります
	一般某处有某物都用 机に　本が　あります。
	为什么用が　，写真展 是一个表示动作的活动，类似的词语还有 結婚式　会議　展覧会

8. いかがですか
	是 どうですか 的礼貌用法
	コーヒーは　いかがですか。（你喝咖啡吗？）
9. 気に　入りました　和 好きです
	
	気に　入りました 相当于汉语中的喜欢，但不表示自己原有的爱好，仅用于第一次见到某物而感到中意，其对象用 が　表示，这种说法只能用于朋友之间，不能用于长辈。
	好きです　一般用于表示持续性爱好
